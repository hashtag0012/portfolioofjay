# Jay's Portfolio - Premium Design

## Overview
A stunning, award-worthy portfolio website for Jay featuring advanced animations, custom cursor, loading screen, and professional design. Built with React, Vite, and Framer Motion.

## Tech Stack
- **Frontend**: React 18 + Vite
- **Animations**: Framer Motion (advanced)
- **Icons**: Lucide React
- **Fonts**: Space Grotesk, Syne, Instrument Serif
- **Port**: 5000

## Premium Design Features
- Custom animated cursor with hover states
- Cinematic loading screen with name reveal
- Scroll progress indicator
- Ambient floating gradient lights
- Film grain texture overlay
- Parallax scroll effects
- Glassmorphism cards
- Magnetic button effects
- Smooth text reveal animations
- Responsive navigation with active states

## Project Structure
```
client/src/
├── components/
│   ├── CustomCursor.jsx   # Interactive cursor
│   ├── Loader.jsx         # Loading animation
│   ├── Navigation.jsx     # Fixed nav with scroll effects
│   ├── Hero.jsx           # Animated hero section
│   ├── Services.jsx       # 6 service cards
│   ├── Portfolio.jsx      # Project showcase
│   ├── WhyMe.jsx          # Trust section + testimonial
│   ├── Contact.jsx        # CTA with methods
│   └── styles.css         # All component styles
├── App.jsx                # Main app with layout
├── App.css                # Global styles + animations
└── index.css              # Base imports
```

## Running
```bash
cd client && npm run dev
```

## Customization
- Contact: Update links in `Contact.jsx`
- Projects: Add real images in `Portfolio.jsx`
- Services: Modify in `Services.jsx`
- Stats: Update in `Hero.jsx`
- Testimonial: Edit in `WhyMe.jsx`
